# 📅 Day 3 - Tokens & Response Usage

## 📖 What I Learned

### 🔹 Tokens
- Tokens are the units of text processed by an AI model.
- Both the input (prompt) and output (response) are measured in tokens.

### 🔹 Prompt Tokens
- Tokens used by the input prompt.
- Longer prompts consume more prompt tokens.

### 🔹 Completion Tokens
- Tokens generated by the AI in its response.
- Longer responses use more completion tokens.

### 🔹 Total Tokens
- Total tokens used in a request.
- **Formula:**
  ```
  Total Tokens = Prompt Tokens + Completion Tokens
  ```

### 🔹 Max Tokens
- Sets the maximum number of tokens the model can generate in the response.
- Helps control response length and token usage.

### 🔹 Finish Reason
- Indicates why the model stopped generating the response.

Common values:
- `stop` → Response completed normally.
- `length` → Stopped because the maximum token limit was reached.
- `tool_calls` → Stopped to call a tool (if supported by the model).

---

## ✅ Key Takeaways
- Every API request consumes tokens.
- Prompt Tokens represent the input.
- Completion Tokens represent the output.
- Total Tokens = Prompt Tokens + Completion Tokens.
- `max_tokens` limits the response length.
- `finish_reason` explains why generation stopped.
