# 📅 Day 2 - System Role & Temperature

## 📖 What I Learned

### 🔹 System Role
- Defines the behavior or personality of the AI model.
- Helps guide the type of responses the model should generate.
- Example:
  > *"You are a brand manager who suggests names for my food company."*

### 🔹 Temperature
- Controls the creativity of the model's responses.
- **Range:** `0.0` to `2.0`
- **Default Value:** `0`

| Temperature | Behavior |
|-------------|----------|
| `0.0` | More deterministic, consistent, and factual responses. |
| `0.5 - 1.0` | Balanced between creativity and accuracy. |
| `1.5 - 2.0` | More creative, diverse, and sometimes unpredictable responses. |

## ✅ Key Takeaways
- System Role defines **how the AI should behave**.
- Temperature defines **how creative the AI's responses should be**.
- Higher temperature = More creativity.
- Lower temperature = More consistent and predictable responses.
